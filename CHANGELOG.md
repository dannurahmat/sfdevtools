# Changelog

All notable changes to the **SF DevTools** extension will be documented in this file.

## [0.0.1] - 2026-01-02

### Added
- Initial release of SF DevTools.
- **Metadata Browser**: Explore and manage metadata components.
- **SOQL Builder**: Interface for building and executing SOQL queries.
- **Tooling API Query**: Support for querying the Salesforce Tooling API.
- **GraphQL Explorer**: Interactive explorer for Salesforce GraphQL.
- **Apex Log Viewer**: Download and view Apex debug logs.
- **Sidebar Integration**: Custom activity bar icon and views for Tools and Org Connection.
